## 📌 About This Repository

This repository contains **well-structured, clean solutions** to the famous **[Striver A2Z DSA Sheet](https://takeuforward.org/strivers-a2z-dsa-course/strivers-a2z-dsa-course-sheet-2/)** — a handpicked roadmap by [Striver (Raj Vikramaditya)](https://www.youtube.com/@takeUforward) to help you go from **zero to hero** in DSA, and crack placements at top product-based companies.

Whether you're a **beginner** starting your DSA journey or a **final-year student** prepping for campus placements — this repo is for **you**.

### ✨ What makes this repo useful?
- 📂 **Topic-wise organized** solutions — easy to navigate
- 💬 **Commented code** for better understanding
- ⚡ **Multiple approaches** wherever possible (Brute → Better → Optimal)
- 🌐 Solutions in **C++** (primary) with Python alternatives where helpful
- 🔄 **Actively maintained** and growing with new solutions regularly

## 🤝 How to Contribute

Contributions are what make this repository truly valuable for the community! If you have a better solution, an alternative approach, or want to add a missing problem — feel free to contribute.

### 📋 Steps to Contribute

1. **Fork** this repository
2. **Clone** your fork:
   ```bash
   git clone https://github.com/<your-username>/DSA-Placement-Prep.git
   ```
3. **Create a branch** for your contribution:
   ```bash
   git checkout -b add/topic-problemname
   ```
4. **Add your solution** following the file naming and comment format shown above
5. **Commit** your changes:
   ```bash
   git commit -m "Add: [Topic] Problem Name - Approach"
   ```
6. **Push** to your fork:
   ```bash
   git push origin add/topic-problemname
   ```
7. **Open a Pull Request** — I'll review and merge it! 🎉

### ✅ Contribution Guidelines
- Follow the existing folder and file naming convention
- Include problem link and approach explanation in comments
- Make sure your code compiles and runs correctly
- One problem per file, please
- Avoid plagiarism — write your own solutions
